Title

O elemento title define o título do documento.

O elemento title é necessária em todos os documentos HTML/XHTML.

O elemento title define um título na aba do navegador.
Fornece um título para a página quando ele é adicionado aos favoritos.
Exibe um título para a página de resultados de pesquisas.

Style

O elemento style é usado para definir as informações de estilo para um documento HTML.

Link

O elemento link define a relação página para um recurso externo como um css ou um javascript conectando as folhas de estilo.

Meta

O elemento meta é utilizado para especificar descrição da página, palavras-chave, autor, e outros metadados.

Os metadados são usados ​​pelos navegadores (como exibir conteúdo), por motores de busca (palavras-chave), e outros serviços da web.

Scripts

O elemento script é usado para definir JavaScripts do lado do cliente.

Base

O elemento base especifica a URL base e destino de base para todas as URLs relativas em uma página
